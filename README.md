# Harini Naraharisetti - Personal Portfolio Website

A premium, highly interactive, and responsive personal portfolio website built with HTML, CSS (Vanilla), and JavaScript. This portfolio showcases academic background, data science skills, hackathon achievements, and project works for Naraharisetti Harini.

## ✨ Features

- **Cinematic Preloader**: A smooth dual-ring loading animation that fades out when the page loads, customized to "HARINI PORTFOLIO".
- **Premium Dark Theme**: Sleek glassmorphic details and a refined dark UI.
- **Interactive Lightbox Modal**: Click on any project card to view high-resolution custom mockups, categories, and descriptions.
- **Dynamic Timeline**: Elegant timeline layouts for Education background and Hackathon achievements.
- **AJAX Contact Form**: Fully integrated with Web3Forms to deliver user inquiries straight to Harini's inbox.
- **Floating Back-to-Top Button**: Smooth scrolling navigation for easy browsing.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile views.

## 🚀 Getting Started

### Prerequisites

You only need a modern web browser to open the site.

### Running Locally

1. Clone or download the repository.
2. Open `index.html` directly in your browser, or run a local web server (e.g. VS Code Live Server).

## 🛠️ Built With

- **HTML5** (Semantic structure)
- **CSS3** (Custom transitions, flexbox, grid, glassmorphism)
- **JavaScript** (Page tabs, details modals, form validation, AJAX submission)
- **Web3Forms API** (Email delivery backend)
- **Ionicons** (Custom UI icons)
